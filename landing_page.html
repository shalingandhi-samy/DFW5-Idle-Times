What it does

Everything you need to track idle time at a glance.

⚡

Live DRAX Data

Scrapes drax.walmart.com/associates/ every 5 minutes. Always current, no manual refresh needed.

🎯

Filtered by Role

Shows only Stationary Picking (019209516) and Box Finishing (019034514) associates with Status: IN.

⏱️

Live Idle Timers

Ticker counts up from each associate's last DRAX scan timestamp — updates every second in the browser.

📊

Sorted by Idle Time

Longest idle associate always floats to the top. Re-ranks automatically on every scrape.

🚦

Color-Coded Alerts

Green <15 min · Yellow 15–29 min · Red 30+ min. Instant visual priority.

Get it running

Four steps from clone to live dashboard.

1

Clone the repo

git clone https://github.com/shalingandhi-samy/DFW5-Idle-Times.git
cd DFW5-Idle-Times
2

Create venv & install deps

uv venv .venv
.venv\Scripts\activate
uv pip install fastapi "uvicorn[standard]" jinja2 playwright
3

Run it

python main.py

First run will pop an Edge window for DRAX SSO login — after that it's headless.

4

Open the dashboard

http://127.0.0.1:8001
